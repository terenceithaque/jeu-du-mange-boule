Le jeu du mange boule, qu'est-ce-que c'est ?

Le jeu du mange boule est un jeu de mon imagination écrit en Python. Vous y jouez un Pac-man qui doit manger des boules réparties sur un canevas.

Consulter la documentation de la construction du jeu. (Fichier .odt)

Les prérequis pour jouer

Pour jouer, vous aurez besoin des logiciels/modules suivants :

- Le langage de programmation Python en version 3 (si possible 3.10) installé sur votre système. Si vous ne possédez pas le logicel, vous pouvez le téléchager sur python.org pour télécharger la version correspondant à votre système d'exploitation (Windows,macOS, GNU/Linux, autres).

- Le module complémentaire keyboard pour Python. Vous pouvez l'installer de 2 manières différentes :

- Sur le site Pypi.org

-En ligne de commande. Pour cela, vous devez configurer le programme pip (fourni avec Python)

Configurer le programme pip (page en anglais).
